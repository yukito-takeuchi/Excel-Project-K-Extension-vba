## 3. 残タスク一覧（4/27打ち合わせまで）

### 🔴 必須タスク

#### T-1：シートインデックス確認（クライアント実機）
- **内容：** `0全科目月次ﾃﾞｰﾀ出力.xlsm` を開き、シートタブのインデックスを確認
- **確認事項：**
  - Sheet(1) = 三期分データシート
  - Sheet(4) = 602当期のみシート
  - Sheet(6) = 税込データ専用シート
- **手順：** Excel → シートタブ右クリック →「シートの移動またはコピー」でインデックス確認
- **対応：** インデックスが異なる場合は `CSV自動取込.bas` の定数値を修正

#### T-2：マクロ動作確認チェックリスト（クライアント実機）
- [ ] 税抜方式：当期CSV → Sheet(4) に正しく貼り付けられるか
- [ ] 税抜方式：税込CSV → Sheet(6) に正しく貼り付けられるか
- [ ] 税込方式：当期CSVのみ取り込み、Sheet(6) は変更されないか
- [ ] 三期分スキップ：Sheet(1) が変更されないか
- [ ] キャンセル時：データが消えないか
- [ ] 完了後：取り込み行数が表示されるか

#### T-3：FreeWay出力CSVの文字コード確認
- **確認内容：** FreeWayが出力するCSVがShift-JISかUTF-8か
- **対応：** Shift-JIS → 現行コードで対応OK / UTF-8 → A-10を実施

#### D-1：打ち合わせ用デモ資料準備
- デモシナリオ（操作順序・説明ポイント）
- ブラウザでdemo/index.htmlを開いてデモ実演
- よくある質問・回答想定集

#### D-2：見積書・料金提示
- 最終フェーズ（テスト・調整・納品）の見積もり
- 追加オプション料金表（A-10, A-11）

- **条件：** T-3でUTF-8と判明した場合のみ

## 8. デモ実演シナリオ（打ち合わせ用）

### 実演の流れ（約10〜15分）

```
[1] SCREEN 0：CSV取込画面を見せる
    → 「毎月これをクリックするだけで完了します」

[2] 税込/税抜切替ボタンで方式を選択
    → 「FreeWayの設定に合わせて切り替えます」

[3] CSVを3つ選択 →「▶ 取り込み → 全レポート生成」をクリック
    → プログレスバーで処理進行を見せる

[4] SCREEN 1b〜1g：各レポートが順番に自動生成される様子を見せる
    → 表紙・月別推移・三期比較・分析表・経営計画・CF推移

[5] SCREEN 2・3：CF計算書・決算書図表の生成
    → 「既存のマクロもそのまま動きます」

[6] SCREEN 4（完了画面）
    → 「9ファイルすべて自動生成完了。PDFとして保存も可能です」
```

### 想定Q&A

| 質問 | 回答 |
|-----|------|
| 既存のExcelファイルは変わらない？ | はい。`0全科目`ファイルにのみマクロを追加。他は変更なし |
| 毎月何分かかっていたのが何分になる？ | 目安：手動15〜30分 → マクロ2〜3分 |
| FreeWayのCSVをそのまま使える？ | はい。FreeWayの標準出力CSVをそのまま選択するだけ |
| 間違えてクリックしたら？ | キャンセル・スキップ機能あり。確認ダイアログで安全に中断可能 |
| 他の会計ソフトにも対応できる？ | CSV形式であれば対応可能（要カスタマイズ） |
