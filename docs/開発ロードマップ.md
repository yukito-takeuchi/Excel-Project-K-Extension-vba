## 残タスク詳細

### 🔴 MUST（必須・期限まで必ずやる）

#### W-1：VBAを0番ファイルにインポート（Windows作業・1〜2時間）

**手順：**
1. `0全科目月次データ出力.xlsx` を開く（まだ .xlsm でない場合は名前を付けて保存 → Excel マクロ有効ブック）
2. `Alt + F11` でVBエディタを開く
3. `ファイル → ファイルのインポート` → `CSV自動取込.bas` を選択
4. インポートされたことを確認（左のツリーに「CSV自動取込」モジュールが表示されるか）
5. `Ctrl + S` で保存

#### W-2：マクロ実行ボタンをシートに設置（Windows作業・30分）

**手順：**
1. 「必須入力」シートを開く
2. `開発` タブ → `挿入` → `ボタン（フォームコントロール）`
3. シート上にドラッグして描画
4. マクロの割り当て → `CSV取込` を選択 → OK
5. ボタンのテキストを「▶ CSV自動取込」に変更
6. ボタンの位置・サイズを整える

#### T-1：シートインデックスの確認（Windows作業・15分）

**確認内容：** 0番ファイルのシートが以下の順番かを確認する

| インデックス | 期待するシート名 |
|------------|----------------|
| Sheet(1) | 602全科目月次データ出力（三期分） |
| Sheet(4) | 602全科目月次データ出力（当期のみ） |
| Sheet(6) | 税込データ専用 |

**確認方法：**
- VBエディタで `?ThisWorkbook.Sheets(1).Name` をイミディエイトウィンドウで実行
- または「ホーム」タブ → シートタブを右クリック → 「シートの移動またはコピー」で順番確認

**不一致の場合：** `CSV自動取込.bas` の定数を修正する
```vba
Private Const IDX_三期分 As Integer = 1  ' ← 実際の番号に変更
Private Const IDX_当期   As Integer = 4  ' ← 実際の番号に変更
Private Const IDX_税込   As Integer = 6  ' ← 実際の番号に変更
```

#### T-2：動作テスト（Windows作業・1〜2時間）

テストするシナリオ：

| # | シナリオ | 期待動作 | 結果 |
|---|---------|---------|------|
| 1 | 税抜方式・全CSV取り込み | 3シートに正しく貼り付け | □ |
| 2 | 税抜方式・三期分スキップ | Sheet(1)は変更されない | □ |
| 3 | 税抜方式・税込CSVスキップ | Sheet(6)は変更されない | □ |
| 4 | 税込方式 | Sheet(6)に貼り付けない | □ |

| 6 | 完了後のメッセージ | 取り込み行数が表示される | □ |
| 7 | 空CSVを選択 | エラーメッセージが出る | □ |

#### T-3：文字コードの確認（Windows作業・15分）

FreeWayのCSV出力が **Shift-JIS** か **UTF-8** かを確認する。

**確認方法：**
1. FreeWayからCSVをエクスポート
2. メモ帳で開く → 文字化けしていなければ Shift-JIS（現行マクロで問題なし）
3. 文字化けする場合 → UTF-8 → A-10（UTF-8対応修正）が必要

### 🟡 SHOULD（できれば・余裕があれば）

#### O-1：既存マクロの一括実行ボタン追加（要クライアント確認後）

0番ファイルに「▶ 全レポートをまとめて更新」ボタンを追加。
各番号ファイルのマクロを `Application.Run` で順番に呼び出す。

**実装例（5〜10分）：**
```vba
Sub 全レポート一括更新()
    Application.Run "'4月次収益分析表BS経営分析表.xlsm'!ストラック図作成"
    Application.Run "'5三期比較決算図表.xlsm'!三期連続比較表作成"
    Application.Run "'12決算書図表.xlsm'!決算書図表作成"
    ' ... 必要なファイルを追加
End Sub
```

> ⚠️ 各マクロが「ラベル位置を調整してください」MsgBoxを出すため、
> 完全自動化は構造上不可能。一括実行しても手動確認は残る。

#### O-2：UTF-8対応（T-3でUTF-8と判明した場合のみ）

```vba
Set csvWb = Workbooks.Open(csvPath, , , , , , , xlWindows)
```
`xlWindows` → Shift-JIS、`65001` → UTF-8 で開くオプションを追加。
