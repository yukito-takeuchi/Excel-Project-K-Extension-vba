# 財務レポート自動化システム　開発提案書

---

## 1. プロジェクト概要

FreeWay（会計ソフト）から出力したCSVデータを、複数のExcelファイルへ自動取り込みし、各種財務グラフを生成するシステムへのマクロ追加開発。

現在、毎月手作業で行っているCSV貼り付け作業をワンクリックで完結させることで、**作業時間の削減とヒューマンエラーの防止**を実現します。

---

## 2. 現状の課題

| 課題 | 内容 |
| ----- | ----- |
| 手作業による非効率 | CSVを開く → コピー → 対象シートへ貼り付けを毎月繰り返し |
| ミスのリスク | 税抜・税込の貼り付け先を間違えると全グラフに影響 |
| 属人化 | 手順を知っている人しか対応できない |

---

## 3. 実装内容

**機能1：CSV自動貼り付け**
ファイル選択ダイアログでCSVを選ぶだけで、対応するシートへ自動貼り付け。

**機能2：税込 / 税抜の自動振り分け**
- 税抜CSV → 602全科目月次データ出力（当期のみ）
- 税込CSV → 税込データ専用シート

**機能3：3期分CSVのスキップ対応UI**
頻度が低い場合はスキップできるボタンを設置。

---

## 4. 対象ファイル構成

```
0全科目月次データ出力.xlsm　← 今回の実装対象（.xlsxから変換）
├ 1表紙.xlsx
├ 2月別推移グラフ.xlsx
├ 3売上三期比較グラフ.xlsx
├ 4月次収益分析表･BS～経営分析表.xlsm
├ 5三期比較決算図表.xlsm
├ 6経営計画･予実グラフ.xlsm
├ 10キャッシュ推移表～運転資金倍率表.xlsm
├ 11キャッシュフロー計算書.xlsm
└ 12決算書図表.xlsm
```

**重要制約：既存ファイルは一切変更しない。0ファイルのみに実装する。**

---

## 5. 工数・期間・費用

| フェーズ | 内容 | 期間 |
| ----- | ----- | ----- |
| フェーズ1 | 全ファイル解析・構造把握 | 3〜4週間 |
| フェーズ2 | マクロ実装・開発 | 1週間 |
| フェーズ3 | テスト・修正・調整 | 2週間 |
| **合計** |  | **約2ヶ月** |

**お見積り：50〜80万円（税別）**

※ファイル間の数式連鎖の複雑度により変動する場合があります。
※フェーズ1（解析）のみ先行契約も可能。

---

## 6. リスクと対策

| リスク | 対策 |
| ----- | ----- |
| 数式連鎖が予想以上に複雑 | フェーズ1を別契約にして先に調査 |
| 貼り付け位置のズレで全ファイルに影響 | テスト環境で十分に検証後リリース |
| 既存マクロへの予期せぬ影響 | 既存ファイルには一切手を加えない方針 |

---

## 7. 進め方（推奨）

解析フェーズのみ先行して別契約にすることを推奨。構造把握が完了した段階で実装フェーズの見積もりをより正確に提示できる。リスクを最小化しながら安全に進められる。
